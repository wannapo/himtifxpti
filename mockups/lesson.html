Pengenalan HTML

HTML (HyperText Markup Language) adalah bahasa standar yang digunakan untuk membuat halaman web. Setiap halaman web yang Anda lihat di internet dibangun menggunakan HTML.

Apa itu HTML?

HTML bukan bahasa pemrograman — melainkan bahasa markup. HTML menggunakan "tag" untuk menandai elemen-elemen di halaman web, seperti heading, paragraf, link, gambar, dan lainnya.

Struktur Dasar

Setiap dokumen HTML memiliki struktur dasar:

  • <!DOCTYPE html> — Deklarasi tipe dokumen
  • <html> — Elemen root
  • <head> — Metadata halaman
  • <body> — Konten yang terlihat

Tantangan

Di editor sebelah kanan, coba buat sebuah heading <h1> dan paragraf <p>. Lihat hasilnya di preview secara real-time!

💡 Tips: Gunakan tombol tab pada editor untuk berpindah antara file HTML, CSS, dan JavaScript.
